# 数据集介绍
首先在该目录下创建三个空目录：train, val和test，同时在三个空目录分别创建子目录img和gt，前者存储原始图像，后者存储分割后得到的二值化图像。

然后根据以下链接下载数据集，并自行放在该目录下。每个文件夹均同时包含img和gt的子目录，以保证代码正常执行。
